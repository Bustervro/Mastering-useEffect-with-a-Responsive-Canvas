# Mastering-useEffect-with-a-Responsive-Canvas
# Responsive Card useEffect Assignment

This React project uses the `useEffect` hook to track the browser window size in real time.

The app displays the current window width and height. It also changes the card layout and background color depending on whether the screen is mobile or desktop size.

## Features

- Tracks `window.innerWidth`
- Tracks `window.innerHeight`
- Uses `window.addEventListener("resize")`
- Removes the event listener with cleanup
- Changes layout for mobile and desktop screens
